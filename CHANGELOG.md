# Changelog

## [1.0.0-beta7] - 2026-03-07

### Changed

- **统一图像保存位置** - 生成结果卡片上的保存按钮现在会使用你在设置中配置的保存目录，和自动保存路径保持一致。
- **改进元数据兼容性** - 新保存图片的元数据结构已对齐 NovelAI 官方格式，网页解析识别成功率更高。

### Fixed

- **修复发布被远端拒绝的问题** - 构建产物 ZIP 不再进入仓库历史，发布流程更稳定。

## [1.0.0-beta5] - 2026-03-01

### Added

- **全局字体大小调整** - 在设置 > 外观中新增字体大小调节选项（70%-150%），支持实时预览和对话框精细调节
- **自动更新检查** - 添加应用更新检查功能，支持检测新版本并在设置页面显示更新提示

## [1.0.0-beta4.3] - 2026-03-01

### Fixed

- **修复窗口状态保存** - 修复窗口状态保存日志格式问题
- **修复背景色** - 修正窗口背景色显示

## [1.0.0-beta4.2] - 2026-03-01

### Added

- **词库分组视图** - 全新的分组视图模式，按类别组织词库条目，支持升序/降序排序
- **词库预览图自定义** - 支持调整词库条目的预览图显示范围，包括偏移和缩放控制，提供可视化裁剪对话框
- **词库 Toolbar 改造** - 新增视图切换按钮（平铺/分组/紧凑）和排序下拉菜单，操作更便捷
- **悬浮按钮动效** - 词库条目卡片悬浮按钮添加平滑动画和 Tooltip 提示

### Changed

- **优化快捷键处理** - 统一快捷键处理逻辑，修复快捷键配置变化后未能正确重建的问题
- **生成页面组件化** - 重构桌面端生成页面布局，提取为独立的可复用组件
- **词库条目卡片交互优化** - 改进背景与边框层级，修复悬浮按钮事件响应问题

### Fixed

- **修复 Sliver 布局错误** - 解决 SliverPersistentHeader 的 layoutExtent 超过 paintExtent 的报错
- **修复画廊路径冲突** - 解决图片移动后路径唯一约束冲突问题
- **修复预览图布局** - 修复 ThumbnailDisplay 和 ThumbnailCropDialog 的布局和缩放问题
